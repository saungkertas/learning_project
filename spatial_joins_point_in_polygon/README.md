# Spatial Joins & Point-in-Polygon (Day 2)

## Overview
Demonstrates GeoPandas spatial joins:
1. Sample coffee shop points.
2. Neighborhood polygons.
3. gpd.sjoin() to assign points to polygons ("within").
4. Aggregate shop_count per neighborhood.
5. Export GeoJSON for geojson.io.

## Run
1. `pip install geopandas matplotlib folium`
2. `jupyter notebook spatial_joins_pip.ipynb`
3. Open `neighborhoods_shops.geojson`.
