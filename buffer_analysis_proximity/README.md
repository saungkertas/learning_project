# Buffer Analysis & Proximity (Day 3)

## Overview
Demonstrates GeoPandas buffering:
1. Sample subway stations (points).
2. Reproject to UTM (meters).
3. 500m buffer around stations.
4. Reproject back to WGS84 (EPSG:4326).
5. Export GeoJSON for geojson.io.

## Run
1. `pip install -r requirements.txt`
2. `jupyter notebook buffer_analysis.ipynb`
3. Open `subway_buffers.geojson`.
